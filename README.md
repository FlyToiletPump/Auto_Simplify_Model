# Auto_Simplify_Model
### 生成式3D模型的自动减面研究。复现传统QEM算法对生成式3D模型的自动减面基线，分析评测输出的结果，并在其基础上引入改良后的QEM算法，映入深度学习神经网络的技术，实现高效、选择性的网格塌缩
一、 核心课题解读与关键点分析
课题本质：研究如何对由生成式AI（如NeRF、Mesh生成网络）创建的高精度3D网格模型，进行高质量的自动简化（减面/网格简化），并满足一系列严格的工业应用级技术指标。
核心矛盾与创新点：
传统减面算法（参考文献[5] Garland的QEM）针对的是手工建模的、干净规则的网格。而生成式模型（如Meshformer、NeRF）输出的网格常带有不规则拓扑、噪声、非流形结构、过度细分或特征模糊等问题。直接将传统算法用于此类网格，效果往往很差。因此，需要“因地制宜”，要么改进传统算法使其适应生成模型，要么引入新的（如深度学习）方法。

二、 技术路线与实施方案建议
 “数据预处理 -> 特征增强减面 -> 后处理与评估” 的三段式技术路线，并融入深度学习以应对复杂工程问题。

阶段一：数据准备与问题分析 (对应复杂问题(1)(2))
1. 数据集选择与处理：
首选推荐：Thingi10K。 它包含大量用户上传的、质量不一的模型，更贴近“生成模型可能输出的不完美网格”这一场景，挑战性最大，研究价值也最高。
次选：ShapeNetCore。 类别清晰，模型相对规范，适合做基线测试和原理验证。
数据处理： 必须编写健壮的数据清洗管道，包括：修复非流形边、去除孤立顶点、统一法线方向、网格重划分。这是后续所有工作的基础。
2. “生成模型”的模拟： 由于直接使用生成模型输出作为输入不稳定，建议采用 “高模 -> 加噪/重划分 -> 作为‘生成模型输出’” 的模拟流程。例如，对Thingi10K中的干净模型进行表面细分或引入轻微几何噪声，来模拟生成模型可能存在的缺陷。

阶段二：核心减面算法设计与实现 (对应复杂问题(1)(3)(4)(5)(7))
这是毕业设计的核心。建议采用 “基于深度特征感知的渐进式网格简化” 框架。
1. 基线算法实现（必做）：
首先复现经典算法（如QEM）。用它处理数据集，分析其在“生成式模型”上的失败案例（如折痕模糊、特征消失），这将清晰论证引入新方法的必要性。
2. 创新算法设计（关键）：
思路：将深度学习作为“特征感知器”，指导传统的边折叠/顶点删除决策。
具体方案：
特征提取： 使用一个轻量级的图神经网络或Mesh CNN，输入模型局部邻域（如以顶点为中心的k-ring面片），学习并输出每个顶点或边的“特征重要性分数”。重要特征包括：折痕边、角点、高频纹理区域、语义部件边界等。
成本函数改进： 将传统QEM的几何误差度量 Cost_geom 与神经网络预测的特征重要性权重 Weight_feat 相结合，形成新的边折叠成本： Cost_new = Cost_geom * (1 - α * Weight_feat) 其中 α 是调节参数。这样，高特征重要性的边会被赋予更低的折叠优先级，从而被保留。
渐进式LOD生成： 基于上述成本函数，执行边折叠队列操作，持续记录折叠序列。通过反转此序列，可以轻松实现从低模到高模的平滑过渡（Progressive Mesh），天然满足“任意两个邻近LOD间平滑转换”的要求。
3. 折痕（Crease）特异性处理：
在数据预处理时，使用基于二面角阈值的算法检测出初始折痕边，并为其打上标签。
在训练特征感知网络时，将“是否为折痕边”作为一个重要的监督信号。
在减面过程中，对标记为折痕的边施加额外的保护权重，确保其保留率 ≥ 80%。

阶段三：实验验证与系统展示 (对应复杂问题(6)(7))
1. 实验平台构建：
开发环境： Python + PyTorch。3D可视化与处理库：Open3D, Trimesh, PyVista。深度学习网格库：Pytorch3D, MeshCNN。
评估指标编程实现：
Hausdorff距离 & RMSE： 使用 Open3D 或 MeshLab 的命令行工具进行精确计算，保证重建误差 ≤ 0.5%（需明确定义，如以模型包围盒对角线长度的百分比为基准）。
视觉质量主观评分： 设计在线评分网页或本地GUI，让至少20名参与者对减面前后模型进行对比评分（百分制），取平均分确保 >80分。
折痕保留率： （减面后保留的折痕边数量 / 原始折痕边数量）* 100%。
2. 效果展示：
可视化对比： 制作并排对比图、误差热力图（用颜色映射几何误差）。
领域应用演示： 将不同LOD的模型导入 Unity 或 Unreal Engine，制作一个简单的场景，通过滑动条动态切换LOD，展示其在游戏/VR中节省性能的效果。录制演示视频。
